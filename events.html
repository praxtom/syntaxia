  • Society

    There shall be multiple groups who will be given different set of technologies in every round.

    They have to think of an idea to make a project that serves the society using those set of technologies in the best possible way.

    The one who comes up with an idea which is the best amongst the rest(On the basis of cost of implementation, feasibility, practicality and usefulness) wins the round.

    Upon winning the round, the winner of the round gets an option to either reserve a cash prize for themselves or buy themselves and extra 15 minutes of time for the next round to think better. Example : On winning a round, the group can reserve an extra Rs.100 to the total cash prize. So, if the club wins the first round, and if they lose the next one, they at least take the reserve Rs.100 with them. This can be done in the first or second round.
    OR
    The winning group may choose to take an extra 15 minutes to think of an idea by taking more time in the next round which is better than the others as they have more time to brainstorm.

    Different sets of technologies :
    Round one : Cloud computing and IOT
    Round two : Cyber security and cameras
    Round three : Websites and mobile devices

    For more details, please contact : Prakhar Tomar(7022814452)

  • Design.com

    Each team will be given instructions on the spot. Each team has to design the website using resources mentioned below.

    In the first round each team will have to create a website with the mentioned elements

    The emphasis of the competition is to create a original website. Therefore, any team found to be plagiarising will be disqualified

    Each website must not contain material deemed objectionable(e.g. No pornography, no profanity, nothing offensive to a person's gender, sexuality, ethnicity, religious beliefs, no reference to alcohol, tobacco, or drugs)

    The teams who submit their entries within the given two hours from the commencement of the completion will be considered. The code has to be sent along with completed pages

    For the second round, a topic will be given on the spot. Each team will have to design a website based on the topic.

    The website will be judged based on creativity, professional looking code and responsiveness. The website will be tested on multiple browsers.

    Resources : HTML, CSS, JavaScript, Bootstrap (No frameworks to be used)
    Mode of submission : Github or Google Drive

    Duration : Round 1 : 2 hours and Round 2 : 3 hours
    Number of team members : 2 per team
